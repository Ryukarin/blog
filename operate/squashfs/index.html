# 提取 SYSTEM 文件

系统： Ubuntu 18.04 LTS

  1. 首先下载 EmuELEC 4.2 镜像包 (opens new window)

  2. 解压镜像包。右击下载好的镜像包，选择快捷菜单 提取到此处；或者用下面命令解压缩

    kei@kei-KVM:~$ gunzip EmuELEC-Amlogic.aarch64-4.2-Generic.img.gz
    1

  3. 右击解压出来的镜像 EmuELEC-Amlogic.aarch64-4.2-Generic.img 文件，选择快捷菜单的 用 磁盘映像挂载器 打开 进行挂载，会在桌面显示挂载的盘（STORAGE 和 EMUELEC）

  4. 进入已挂载的 EMUELEC 盘

  5. 复制 SYSTEM 文件到桌面

# 查看 SYSTEM 文件类型

使用 file 命令查看文件的文件类型，以便使用对应的命令工具去执行操作。

root@kei-KVM:~# file SYSTEM
SYSTEM: Squashfs filesystem, little endian, version 4.0, 906421384 bytes, 17904 inodes, blocksize: 524288 bytes, created: Fri Jun 18 21:05:07 2021

# unsquashfs 命令解压解包

解包打包最好切换到 root 用户操作。用 unsquashfs 命令解压缩。

# 切换 root 用户
kei@kei-KVM:~$ su -
root@kei-KVM:~#

# 复制上面解压的 SYSTEM 文件到当前位置
root@kei-KVM:~# cp /home/kei/桌面/SYSTEM .

# unsquashfs 命令解包 SYSTEM 文件
root@kei-KVM:~# unsquashfs SYSTEM 
Parallel unsquashfs: Using 2 processors

# mksquashfs 命令压缩打包
