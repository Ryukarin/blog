Excel 函数

karin ... 2021-07-02
  • Excel
 大约 13 分钟
  • 统计函数
  • 逻辑函数
  • 文本函数
  • 数学函数
  • 查找引用函数
  • 日期时间函数
  • 信息函数

Excel 的常用函数大致可分为以下几大类：

  • 统计函数
  • 逻辑函数
  • 文本函数
  • 数学函数
  • 查找引用函数
  • 日期时间函数
  • 信息函数
  • ......

当然，工作中很多问题不是单单一个函数就可以解决的，而是多个函数的组合才能沟解决问题的，这就要我们灵活的运用函数套嵌，或者需要辅助列等等。同一个问题也不是只有一个方法，可以有多种方法，具体情况具体对待。

# 统计函数

函数 功能
AVERAGE 函数 (opens new window) 返回其参数的平均值
AVERAGEIF 函数 (opens new window) 返回区域中满足给定条件的所有单元格的平均值（算术平均值）
AVERAGEIFS 函数 (opens new window) 返回满足多个条件的所有单元格的平均值（算术平均值）
COUNT 函数 (opens new window) 计算参数列表中数字的个数
COUNTA 函数 (opens new window) 计算参数列表中值的个数
COUNTBLANK 函数 (opens new window) 计算区域内空白单元格的数量
COUNTIF 函数 (opens new window) 计算区域内符合给定条件的单元格的数量
COUNTIFS 函数 (opens new window) 计算区域内符合多个条件的单元格的数量
LARGE 函数 (opens new window) 返回数据集中第 k 个最大值
SMALL 函数 (opens new window) 返回数据集中的第 k 个最小值
MAX 函数 (opens new window) 返回参数列表中的最大值
MEDIAN 函数 (opens new window) 返回给定数值集合的中值
MIN 函数 (opens new window) 返回参数列表中的最小值
SUBTOTAL 函数 (opens new window) 返回列表或数据库中的分类汇总

# 逻辑函数

函数
IF 函数 (opens new window) 指定要执行的逻辑检测
IFS 函数 (opens new window) 检查是否满足一个或多个条件，且是否返回与第一个 TRUE 条件对应的值。
IFERROR 函数 (opens new window) 如果公式的计算结果错误，则返回您指定的值；否则返回公式的结果
IFNA 函数 (opens new window) 如果该表达式解析为 #N/A，则返回指定值；否则返回该表达式的结果
AND 函数 (opens new window) 如果其所有参数均为 TRUE，则返回 TRUE
OR 函数 (opens new window) 如果任一参数为 TRUE，则返回 TRUE
NOT 函数 (opens new window) 对其参数的逻辑求反
TRUE 函数 (opens new window) 返回逻辑值 TRUE
FALSE 函数 (opens new window) 返回逻辑值 FALSE
SWITCH 函数 (opens new window) 根据值列表计算表达式，并返回与第一个匹配值对应的结果。
如果不匹配，则可能返回可选默认值。

# 文本函数

函数
ASC 函数 (opens new window) 将字符串中的全角（双字节）英文字母或片假名更改为半角（单字节）字符
DBCS 函数 (opens new window) 将字符串中的半角（单字节）英文字母或片假名更改为全角（双字节）字符
CHAR 函数 (opens new window) 返回由代码数字指定的字符
CONCAT 函数 (opens new window) 将多个区域和/或字符串的文本组合起来，
但不提供分隔符或 IgnoreEmpty 参数。
CONCATENATE 函数 (opens new window) 将几个文本项合并为一个文本项
DOLLAR 函数 (opens new window) 使用 ￥（人民币）货币格式将数字转换为文本
EXACT 函数 (opens new window) 检查两个文本值是否相同
FIXED 函数 (opens new window) 将数字格式设置为具有固定小数位数的文本
FIND、FINDB 函数 (opens new window) 在一个文本值中查找另一个文本值（区分大小写）
LEFT、LEFTB 函数 (opens new window) 返回文本值中最左边的字符
RIGHT、RIGHTB 函数 (opens new window) 返回文本值中最右边的字符
MID、MIDB 函数 (opens new window) 从文本字符串中的指定位置起返回特定个数的字符
LEN、LENB 函数 (opens new window) 返回文本值中最左边的字符
LOWER 函数 (opens new window) 将文本转换为小写
UPPER 函数 (opens new window) 将文本转换为大写形式
PROPER 函数 (opens new window) 将文本值的每个字的首字母大写
PHONETIC 函数 (opens new window) 提取文本字符串中的拼音（汉字注音）字符
REPLACE、REPLACEB 函数 (opens new window) 替换文本中的字符
SEARCH、SEARCHB 函数 (opens new window) 在一个文本值中查找另一个文本值（不区分大小写）
TEXT 函数 (opens new window) 设置数字格式并将其转换为文本
TEXTJOIN 函数 (opens new window) 将多个区域和/或字符串的文本组合起来，并包括你在要组合的
各文本值之间指定的分隔符。 如果分隔符是空的文本字符串，
则此函数将有效连接这些区域。
SUBSTITUTE 函数 (opens new window) 在文本字符串中用新文本替换旧文本

# 数学函数

函数
ABS 函数 (opens new window) 返回数字的绝对值
INT 函数 (opens new window) 将数字向下舍入到最接近的整数
MOD 函数 (opens new window) 返回除法的余数
PI 函数 (opens new window) 返回 pi 的值
RAND 函数 (opens new window) 返回 0 和 1 之间的一个随机数
RANDBETWEEN 函数 (opens new window) 返回位于两个指定数之间的一个随机数
ROUND 函数 (opens new window) 将数字按指定位数舍入
ROUNDDOWN 函数 (opens new window) 向绝对值减小的方向舍入数字
ROUNDUP 函数 (opens new window) 向绝对值增大的方向舍入数字
SUM 函数 (opens new window) 求参数的和
SUMIF 函数 (opens new window) 按给定条件对指定单元格求和
SUMIFS 函数 (opens new window) 在区域中添加满足多个条件的单元格
SUMPRODUCT 函数 (opens new window) 返回对应的数组元素的乘积和

# 查找引用函数

函数
ADDRESS 函数 (opens new window) 以文本形式将引用值返回到工作表的单个单元格
CHOOSE 函数 (opens new window) 从值的列表中选择值
FILTER 函数 (opens new window) FILTER 函数可以基于定义的条件筛选一系列数据。
INDEX 函数 (opens new window) 使用索引从引用或数组中选择值
INDIRECT 函数 (opens new window) 返回由文本值指定的引用
LOOKUP 函数 (opens new window) 在向量或数组中查找值
VLOOKUP 函数 (opens new window) 在数组第一列中查找，然后在行之间移动以返回单元格的值
MATCH 函数 (opens new window) 在引用或数组中查找值
OFFSET 函数 (opens new window) 从给定引用中返回引用偏移量
SORT 函数 (opens new window) 对区域或数组的内容进行排序
ROW 函数 (opens new window) 返回引用的行号
ROWS 函数 (opens new window) 返回引用中的行数
COLUMN 函数 (opens new window) 返回引用的列号
COLUMNS 函数 (opens new window) 返回引用中包含的列数
TRANSPOSE 函数 (opens new window) 返回数组的转置

# 日期时间函数

函数
DATE 函数 (opens new window) 返回特定日期的序列号
YEAR 函数 (opens new window) 将序列号转换为年
MONTH 函数 (opens new window) 将序列号转换为月
DAY 函数 (opens new window) 将序列号转换为月份日期
TIME 函数 (opens new window) 返回特定时间的序列号
HOUR 函数 (opens new window) 将序列号转换为小时
MINUTE 函数 (opens new window) 将序列号转换为分钟
SECOND 函数 (opens new window) 将序列号转换为秒
TODAY 函数 (opens new window) 返回今天日期的序列号
NOW 函数 (opens new window) 返回当前日期和时间的序列号
WEEKDAY 函数 (opens new window) 将序列号转换为星期日期
WEEKNUM 函数 (opens new window) 将序列号转换为代表该星期为一年中第几周的数字
DATEDIF 函数 (opens new window) 计算两个日期之间的天数、月数或年数。 此函数在用于计算年龄的公式中很有用。
DAYS 函数 (opens new window) 返回两个日期之间的天数
EDATE 函数 (opens new window) 返回用于表示开始日期之前或之后月数的日期的序列号
EOMONTH 函数 (opens new window) 返回指定月数之前或之后的月份的最后一天的序列号
NETWORKDAYS 函数 (opens new window) 返回两个日期间的完整工作日的天数
NETWORKDAYS.INTL 函数 (opens new window) 返回两个日期之间的完整工作日的天数（使用参数指明周末有几天并指明是哪几天）
WORKDAY 函数 (opens new window) 返回指定的若干个工作日之前或之后的日期的序列号
WORKDAY.INTL 函数 (opens new window) 返回日期在指定的工作日天数之前或之后的序列号（使用参数指明周末有几天并指明是哪几天）

# 信息函数
CELL 函数 (opens new window) 返回有关单元格格式、位置或内容的信息
ISBLANK 函数 (opens new window) 如果值为空，则返回 TRUE
ISERR 函数 (opens new window) 如果值为除 #N/A 以外的任何错误值，则返回 TRUE
ISERROR 函数 (opens new window) 如果值为任何错误值，则返回 TRUE
ISEVEN 函数 (opens new window) 如果数字为偶数，则返回 TRUE
ISODD 函数 (opens new window) 如果数字为奇数，则返回 TRUE
ISLOGICAL 函数 (opens new window) 如果值为逻辑值，则返回 TRUE
ISNA 函数 (opens new window) 如果值为错误值 #N/A，则返回 TRUE
ISNUMBER 函数 (opens new window) 如果值为数字，则返回 TRUE
ISTEXT 函数 (opens new window) 如果值为文本，则返回 TRUE
N 函数 (opens new window) 返回转换为数字的值
T 函数 (opens new window) 将参数转换为文本
TYPE 函数 (opens new window) 返回表示值的数据类型的数字

更多的函数分类及其详细用法，请参考官方文档 (opens new window)
